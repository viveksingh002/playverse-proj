## 📌 Overview

**PlayVerse** is a responsive mini-games web application built using **vanilla HTML, CSS, and JavaScript**.
It provides an interactive gaming experience directly in the browser without any installation or login.

This project was created to strengthen core frontend skills such as DOM manipulation, UI design, and game logic implementation.

---

## 🎯 Features

- 🎮 Multiple mini games in one platform
- ⚡ Fast and lightweight performance
- 📱 Fully responsive design (mobile + desktop)
- 🎨 Modern UI with smooth hover effects
- 🧠 Simple and interactive game logic
- 🌐 Fully deployed using GitHub Pages
